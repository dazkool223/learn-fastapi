
## Final Project Challenge

### Build a Complete Student Management System

**Requirements:**
1. **Students**: CRUD operations
2. **Courses**: CRUD operations
3. **Enrollments**: Enroll students in courses
4. **Grades**: Add and update grades
5. **Statistics**: Average grades, top students, etc.

**Bonus Features:**
- Search functionality
- Filter by course/grade
- Export data to CSV
- Email notifications
- Authentication

**Database Schema:**
```
Students: id, name, email, age, enrollment_date
Courses: id, name, code, credits
Enrollments: id, student_id, course_id, enrollment_date
Grades: id, enrollment_id, grade, date
```
